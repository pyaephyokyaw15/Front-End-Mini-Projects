
# Todo List

This is a simple To-Do list app with Vanilla JavaScript.


## Demo

https://chat-to-do-list.netlify.app/

  
## Features

- Add tasks
- Remove taks
- Mark as completed task
- filter Taks
- Sort tasks
- Save data in your browser
