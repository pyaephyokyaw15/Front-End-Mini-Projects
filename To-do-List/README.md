
# Todo List

Simple To-Do List app with Vanilla JavaScript.


## Demo

https://chat-to-do-list.netlify.app/

  
## Features

- Add tasks
- Remove taks
- Mark as completed task
- filter Taks
- Sort tasks
- Save data in your browser(local strorage)
- Compatible with any device(responsive)
